Sobre Mi

Soy estudiante de Informática y Telecomunicaciones en la Universidad Diego Portales, donde estoy aprendiendo diversas tecnologías de desarrollo de software, como PostgreSQL, JavaScript, HTML y CSS, entre otras.

Como proyecto personal, he desarrollado una app web de "cifrado de texto" donde implementé el cifrado César utilizando JavaScript. Esto me permitió comprender cómo utilizar JS para mejorar la navegación de un sitio web y realizar cambios dinámicos en el documento HTML.

Además, he mejorado mis habilidades de forma personal y he obtenido la certificación de "Responsive Web Design" de FreeCodeCamp, lo que me ayudó a comprender mejor el uso de CSS.

Me apasiona el desarrollo de software y el desarrollo web, y también tengo interés en la producción musical y el ejercicio físico.
